# Installation

## Prerequisites

You need to have Node.js / ``npm`` to run the client

## Client Setup Instructions

1. ``git clone LINK``
2. ``cd OpenOrganizer``
3. ``npm install``

### Running and Building

4. ``quasar dev`` to run in the browser
5. ``quasar build -m electron`` for executable
